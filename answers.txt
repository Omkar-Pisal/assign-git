•  What is a gitignore file and what should it include for a Java project? 
gitignore file tells git to ignore certain types of file i.e. '.class' files or '.jar' file etc.
For java project it should include files generated by OS or IDE as well as '.class' files.

•  What does fast-forward in git mean? 
It basically means to merge your changes from one branch to other. It is possible if direct linear path from source branch to target branch is available.

•  Why should you have different branches? 
Not having development branches and developing only on master branch is not a good practice since it could lead to problems later. Having branches and developing new features on it further testing can keep software stable in master while experiment is done in develoment branch. Later if everything looks good, this branch can be merged into master. 

•  In a project how many remote Git repositories do you usually have? 
You can have one or more remote git repositories in a project.
You can add repositiores by using command git remote add <shortname> <url>
